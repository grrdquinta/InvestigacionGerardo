¿Qué son los componentes pasivos en la Red?

Los componentes pasivos de la red son elementos que no requieren una fuente de alimentación y no amplifican ni regeneran las señales. Son esenciales para los bloques de construcción de la red y realizan funciones fundamentales como conectar dispositivos, dividir señales o alojar el equipo de red activo.

Descripción técnica de cada componente pasivo

RJ45

es un conector que se utiliza para conectar a redes dispositivos mediante un cable que puede ser de hasta 8 hilos en su interior, el conector posee estos 8 pines, aunque no siempre se utilizan todos. El conector RJ45 puede ser macho o hembra, aunque lo más común es ver el macho en cables de red de ordenadores, también para conectar a una pared y hacer la conexión más cómoda se puede usar un conector hembra

Jacks o Keystone

Un conector keystone es un conector hembra o un módulo hembra que se puede utilizar en una red de área local (LAN). Un conector trapezoidal generalmente se monta en un panel de pared o panel de conexión.

Last updated 3 mins ago

Cable UTP

UTP son las siglas de Unshielded Twisted Pair (par trenzado no apantallado). Es un cable de red que consta de 8 hilos, divididos en dúos (con los colores azul, verde, naranja y marrón).Hay otros cables para transferir datos, como el cable coaxial o el de fibra óptica, pero los UTP se utilizan habitualmente para las conexiones a Internet.

Last updated 3 mins ago

Patch Panel

Un patch panel , también conocido como panel de conexión o bahía de rutas, es una pieza de montaje de hardware con varios puertos para conectar y gestionar los cables LAN o los cables de fibra/cobre entrantes y salientes. Los paneles de conexión pueden ser bastante pequeños, con sólo unos pocos puertos, o muy grandes, con varios centenares de puertos.

Last updated 3 mins ago

Patch Cord

Los Patch Cords RJ-45 están disponibles en cables UTP, FTP y en longitudes 1 y 2 metros. También versiones especiales de hasta 5 metros. En versiones de PVC y LSOH. La elección entre las 2 versiones, UTP, FTP asegura el apropiado nivel de protección de compatibilidad electromagnética.

Last updated 3 mins ago

Fibra óptica

La fibra óptica es un medio físico de transmisión de información, usual en redes de datos y telecomunicaciones, que consiste en un filamento delgado de vidrio o de plástico, a través del cual viajan pulsos de luz láser o led, en la cual se contienen los datos a transmitir.

Last updated 3 mins ago

Canaletas

Conductos que organizan y protegen cables en instalaciones.

Componentes activos

Son todos aquellos componentes que dependen de una fuente de energía. Por lo general pueden excitar un circuito inyectando energía en el circuito. Estos componentes pueden ser transistores, diodos, amplificadores.

Herramientas para cableado estructurado

Descripción técnica de cada componente pasivo

Ponchadora

Es una herramienta pequeña de diseño ergonómico que cuenta con una forma de alicate o pinza, y se utiliza de manera eficiente para corrugar o crimpar conectores en una variedad de cables por medio de una pequeña deformación que mantiene unida o juntas ambas partes.

Last updated 3 mins ago

Crimpadora

es una versátil herramienta para electricistas, utilizada para cortar cables a la longitud adecuada y crear conexiones seguras y confiables mediante el crimpado de terminales metálicos.

Last updated 3 mins ago

Desforradora

Herramienta utilizada para retirar el recubrimiento exterior de los cables sin dañar los hilos internos.

Last updated 3 mins ago

Tester/probador de cables

es la herramienta ideal para verificar que el armado de cables de red, coaxiales y telefónicos, sea confiable, profesional y de alta calidad.

¿Qué es el cableado horizontal?

El cableado horizontal conecta los dispositivos de red en una misma planta o edificio a través de un panel de distribución.

Tipos de cables para el sistema de cableado horizontal

Existen diferentes tipos de cables para cableado horizontal, como el cable UTP, STP, y fibra óptica, cada uno con características particulares en cuanto a velocidad, interferencias y distancia soportada.

Normativa para hacer cables de red (Clase A y B)

Las normas EIA/TIA-568 definen las categorías A y B para el cableado de red. La diferencia entre ellas radica en el orden de los colores de los pares de cables.

Para qué se utiliza un cable directo en la red

El cable directo se utiliza para conectar dispositivos de red a un switch o hub.

Para qué se utiliza un cable cruzado en la red

El cable cruzado se utiliza para conectar directamente dos dispositivos sin la intervención de un switch o hub.
